# Whisper

#### 介绍

本项目计划开发一个匿名社交网站。社交形式暂定为tape的提问箱模式，即：每位用户可以发起提问箱，其他用户可以以匿名的身份在提问箱中提问，该提问被用户收集后回答。

项目计划开发一个B/S架构的前后端分离的项目。暂定后端使用SSM框架，前端使用Vue框架，数据库使用mySql。
